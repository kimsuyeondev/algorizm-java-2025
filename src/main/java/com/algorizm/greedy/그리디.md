1) 정렬 + 앞에서부터 선택
예: 회의실 배정, 작업 스케줄링

jobs.sort((a,b)->a.end - b.end);
int count = 0, lastEnd = 0;
for (Job j : jobs) {
    if (j.start >= lastEnd) {
        count++;
        lastEnd = j.end;
    }
}
