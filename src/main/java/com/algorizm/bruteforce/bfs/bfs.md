BFS는 “최단 거리 문제”에서 절대적이다
가장 유명한 BFS 문제:
2D 미로에서 (0,0) → (N,M)까지
최소 몇 번 이동해야 하는가?
비용이 똑같아야함 비용이다르면 dp로 넘어간다.
"가까운 곳부터 차례대로 훑는 탐색"
→ 너비/거리 우선
→ 큐
→ 최단 거리, 레벨 탐색
시작점을 큐에 넣고
꺼내서(가장 가까운 노드)
그 주변(이웃)을 큐에 넣고
가까운 것부터 차례대로 탐색
비용이1이어야가능 아닌경우 다익스트라확률

DFS
"끝까지 갔다가 막히면 뒤로 가는 탐색"
→ 깊이 우선
→ 재귀
→ 스택
→ 영역 탐색, 연결된 조각 찾기, 백트래킹
